# SSoT 시스템 헌법 (Single Source of Truth)

이 저장소는 전체 시스템의 가동 루프와 운영 원칙을 규정하는 최상위 가이드입니다.

## 전체 가동 루프
1. **업무 하달**: `.github/ISSUE_TEMPLATE`을 통한 명확한 미션 정의
2. **지침 참조**: `SOP/` 및 `Philosophy/`의 원칙 준수
3. **실행**: `Prompts/` 에이전트를 통한 결과물 생성
4. **회고 및 기록**: `PULL_REQUEST_TEMPLATE`에 따른 실행 결과 리캡

## PR 및 커밋 규칙
- 모든 변경 사항은 PR을 통해 검토됩니다.
- PR 템플릿의 리캡 양식을 반드시 준수합니다.
